Apple Health ❤️
Getting FORGED data into Apple Health
Option 1 — Strong app (easiest)
1. Export CSV from Settings
2. Download Strong (free app)
3. Strong → Profile → Import Workouts → select CSV
4. Strong auto-syncs to Apple Health
Option 2 — Hevy
1. Export CSV from Settings
2. Download Hevy (free app)
3. Hevy → Profile → Import → select CSV
4. Enable Apple Health in Hevy settings
Why no direct connection?
Apple requires native iOS apps to write to HealthKit. FORGED is a web app — bridge apps handle the last step.
